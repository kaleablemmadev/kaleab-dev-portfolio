**Title:** What is a useState in React?

***Voiceover:***
React re-renders (i.e., restarts) UI when state changes. But, what is a state?

You enter your name in the input bar of your google account. Whenever you enter a letter, in react, the state changes from "Joh" to "John". When we enter "n", we are essentially saying to react, "I just changed the state from `Joh` to `John`, so re-render the UI to show the new state.".e@
